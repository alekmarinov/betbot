BetBot - your betting analyzing and advisory tool

-- Install Instructions --

Copy all program files in arbitrary directory, and call betbot.cmd file with full path.
Example: If the program is copied to c:\tools\betbot, you must call c:\tools\betbot\betbot.cmd

If you want to add it to your system PATH and call it with just "betbot" from arbitraty location,
then you must edit betbot.cmd and SET LRUN_HOME to the location where the program is installed.
